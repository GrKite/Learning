### overload的要件是

参数表要不同, 所以这个函数的话, 到底是什么呢?

```c++
#include <iostream>
using namespace std;

class A
{
        int i;
public:
        A()
        :i(0)
        {
        }
        
        ~A()
        {
        }

        void f()
        {
                cout << "f()" << endl;
        }

        void f() const
        {
                cout << "f() const" << endl;
        }
};

int main()
{
        const A a;
        a.f();

        return 0;
}
```

res:

```
f() const
```
