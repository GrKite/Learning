##### 数组的遍历

```c
#include <stdio.h>

int main()
{
	int arr[4] = {1, 2, 3, 4};

	for(int* p = arr; p < arr + 4; p++) // 注： arr + 4
	{
		printf("%d\n", *p);
	}

	return 0;
}
```



#### 小结

> 直接将数组看成一块内存, 数组名就是这块内存的地址, 然后通过指针来访问它就可以了.

- 数组名: 本身就是内存地址, 是数组第一个元素的内存地址
- 数组和指针可以灵活转换
